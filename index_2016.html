Recent Projects

CCM: Cloud Configuration Management System for Elastic Application Deployment in Private Clouds

Demo

01/2015-08/2015 01/2016-08/2016 Credit Suisse NC State University

Cloud infrastructures and distributed applications become increasingly complex, which makes us be thirsting for an easy-to-use cloud application deployment tool. This tool needs elasticity for dynamic environments, for example, it needs to suport geographically distributed hosts, to well-handle cloud system anomalies, and to dynamically change workloads.

The CCM project aims to present this exactly easy-to-use application deployment tool. It has automatic component composition and instantiation. More importantly, it has an elasticaly auto-scaling mechanism to handle overload conditions, resource contentions, and system anomalies.
